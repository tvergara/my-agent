# Bibliographic Audit - Paper 41c60725

**Paper ID:** 41c60725-bb92-47f2-acf8-07f0b99647fb
**Title:** HeiSD: Hybrid Speculative Decoding for Embodied Vision-Language-Action Models with Kinematic Awareness

## Summary of Findings

The bibliography for this submission has a systematic issue with missing capitalization protection for acronyms and model names, which is a common requirement for ICML and other major venues using BibTeX.

### 1. Missing Capitalization Protection for Acronyms
The following acronyms in titles are not protected by curly braces and will likely be lowercased by the bibliography style:
- `VLA` (in `edgevla`, `surveyopenhelix`, `specprune`, `specvla`, `tinyvla`, `efficientvla`, `deervla`, `molevla`, `realtimevla`, `kerv`, `rapid`, `dyq-vla`)
- `LLM` (in `medusa`, `speculative`, `draftandverify`, `HierarchicalDrafting`, `PROMTEC`, `ANPD`)
- `AI` (in `boosting`)
- `CTC` (in `speculative`)
- `SVD` (in `SVD`)
- `PEFT` (in `PEFT-1`, `PEFT-2`, `PEFT-3`)
- `LoRA` / `DoRA` (in `lora-1`, `lora-2`)
- `MoE` (in `moqa`)

While generally better than some other submissions, there are still minor inconsistencies in how conference names are abbreviated or fully expanded (e.g., `CVPR` vs `Proceedings of the Computer Vision and Pattern Recognition Conference`).
