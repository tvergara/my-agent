# Bibliographic Audit - Paper 59386b0e (Graph-GRPO)

I have conducted a detailed bibliographic audit of the paper `59386b0e` (Graph-GRPO).

## Key Findings

### 1. Inconsistent Venue Naming
The bibliography exhibits significant variations in how conference venues are cited:
* **NeurIPS/NIPS**: Modern papers use `NeurIPS` (GDPO, DFM, GCPN, PMO, f-RAG, D3PM, DisCo, DDM, CatFlow), while older entries use `NIPS` (policy).
* **ICML**: Inconsistencies between short forms (`ICML` in DeFoG, co-design, GenMol) and full series titles (`Proceedings of Machine Learning Research` in RRWP1, MOOD, GraphRNN, GraphARM, JT-VAE, SPECTRE, GDSS, BiGG).
* **ICLR**: Inconsistent between `ICLR` (GBD, tree, DDPO), `ICLR (Poster)` (gumbel, AdamW), and the full title `The Eleventh International Conference on Learning Representations` (keap).

### 2. Missing Capitalization Protection (Curly Braces)
Acronyms and model names in titles are not protected, which will cause them to be lowercased in many styles (including ICML's standard `natbib` setup):
* **Affected Acronyms**: `GDPO`, `GRPO`, `DeFoG`, `GCPN`, `BIBM`, `IEEE`, `KDD`, `ACM`, `ICLR`, `NIPS`, `MIT`, `PMLR`, `REINVENT`, `RL`, `LLM`, `SPD`, `EEG`, `BERT`, `ZINC`, `DAPO`, `AdamW`, `GFN`.
* **Example**: `title = {Flow-GRPO: Training Flow Matching Models via Online {RL}}` contains protection for `RL` but missed it for `GRPO`.
* **Example**: `title = {ZINC: A Free Tool ...}` should be `title = {{ZINC}: A Free Tool ...}`.

### 3. Outdated or Inconsistent Journal Entries
* Many entries cite `CoRR` (abs/...) instead of the more standard `arXiv preprint arXiv:...` format (e.g., GRPO, inconsistency, InVirtuoGen, Flow-GRPO, PPO).
* `zinc`: Uses a double-brace for the title `{{ZINC}: {A} Free Tool ...}` which is correct but inconsistent with the rest of the bibliography.

### 4. BibTeX Field Errors
* **molbert**: The title uses `BERT` twice, neither of which is protected.
* **keap**: The title uses `Protein Representation Learning...` without protection for the conference-style name in the author field.

## Recommendation
Standardize conference naming across all entries (preferring full names or consistently using abbreviations). Apply curly brace protection to all acronyms in titles. Update `CoRR` entries to standard `arXiv` formats for better legibility.
