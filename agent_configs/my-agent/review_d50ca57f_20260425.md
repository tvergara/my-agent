# Citation Audit for Paper d50ca57f

**Paper ID:** d50ca57f-ac9a-438f-b0f5-fab02c8d64df
**Title:** Transport Clustering: Solving Low-Rank Optimal Transport via Clustering

## Overview
This audit examines the bibliographic quality of `references.bib` for the paper "Transport Clustering". The analysis focuses on capitalization consistency, venue accuracy, and the status of preprint citations.

## Findings

### 1. Missing Capitalization Protection in Titles
Several titles contain acronyms or specific terms that should be protected with curly braces `{}` to prevent them from being lowercased by the ICML bibstyle.

- **Acronyms:** `NMF` (Algorithms for Non-negative Matrix Factorization), `t-SNE` (Visualizing Data using t-SNE), `LLMs` (Distributional Preference Alignment of LLMs via Optimal Transport), `NVP` (Density estimation using Real NVP).
- **Mathematical Terms:** `K-median` (should likely be `k-median` or protected), `k-means++`.
- **Model/Project Names:** `ImageNet`, `Sinkformers`, `Flow++`, `DeST-OT`, `GiLOT`, `SHAPER`.

### 2. Inconsistent Conference and Journal Names
- **NIPS/NeurIPS Transition:** `NMF` entry uses `Advances in Neural Information Processing Systems 13, Papers from Neural Information Processing Systems {(NIPS)} 2000`. Newer entries use `NeurIPS`. Consistency is preferred.
- **Capitalization:** `vaswani2017attention` uses lowercase `Advances in neural information processing systems`.
- **Publisher Metadata:** Many entries from `Elsevier BV` or `Springer` include publisher information that is usually omitted in conference submissions to save space, but if included, it should be consistent.

### 3. Outdated ArXiv Citations
Several papers are cited as preprints but have been published in formal venues:
- **Manole et al. (2024)** (`manole2024backgroundmodelingdoublehiggs`): Cited as `arXiv:2208.02807`.
- **Sejourne et al. (2022)** (`Sejourne:2022`): Cited as `arXiv:2211.08775`.
- **Lavenant et al. (2021)** (`lavenant2021towards`): Cited as `arXiv:2102.09204`.
- **Halmos et al. (2025)** (`halmos2025hierarchical`): Cited as `arXiv preprint arXiv:2503.03025`.

### 4. Duplicate/Redundant Entries
- `NMF` and `2ef7006f34ff4cd7afa86c9bc8932c80` both refer to "Algorithms for Non-negative Matrix Factorization" by Lee and Seung (NIPS 2000). These should be consolidated.
- `frieze2004fast` and `10.1145/1039488.1039494` both refer to the same JACM paper by Frieze et al. (2004).

## Recommendations
1. Consolidate duplicate entries (`NMF`, `Lee & Seung`).
2. Update ArXiv preprints to their corresponding conference/journal versions where applicable.
3. Protect acronyms (`LLM`, `t-SNE`, `NVP`, `OT`) in titles using `{}`.
4. Standardize venue names (e.g., always use `Advances in Neural Information Processing Systems`).
