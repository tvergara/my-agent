# Bibliographic Audit - Paper acca775c

## Overview
This audit focuses on the citation formatting and bibliographic accuracy of the submission "Expert Threshold Routing for Autoregressive Language Modeling with Dynamic Computation Allocation and Load Balancing" (ID: acca775c). The review identifies several systematic issues in the `example_paper.bib` file that should be addressed to meet professional academic standards.

## Key Findings

### 1. Outdated arXiv Citations
Several foundational and related works are cited as arXiv preprints despite having been formally published in major peer-reviewed venues by early 2026. 

*   **DataComp-LM** (`li2024dclm`): Cited as `arXiv:2406.11794`. This work was formally published in **NeurIPS 2024** (Datasets and Benchmarks Track).
*   **Mixture-of-Depths** (`raposo2024mixture`): Cited as `arXiv:2404.02258`. This work was published in **ICML 2024**.
*   **OLMoE** (`muennighoff2024olmoe`): Cited as `arXiv:2409.02060`. This work was published as an Oral presentation at **ICLR 2025**.

Updating these to their formal conference versions ensures that the bibliography reflects the most current and peer-reviewed sources.

### 2. Missing Capitalization Protection
Technical acronyms and proper nouns in several titles lack curly brace `{}` protection, which will cause them to be incorrectly rendered in lowercase by many BibTeX styles (e.g., "olmoe" instead of "OLMoE").

*   `li2024dclm`: Title contains `DataComp-LM` which should be protected.
*   `muennighoff2024olmoe`: Title contains `OLMoE` which should be protected.
*   `deepseekv3`: Title `DeepSeek-V3 Technical Report` should have `DeepSeek-V3` protected.
*   `shazeer2017outrageously`: Title `Outrageously Large Neural Networks: The Sparsely-Gated Mixture-of-Experts Layer` should likely have `Sparsely-Gated Mixture-of-Experts` or at least the start of terms protected depending on style, but generally acronyms like `MoE` (if used) always need protection.

### 3. Systematic Inconsistencies
*   The use of `arXiv preprint arXiv:...` in the `journal` field is inconsistent with some entries using the `eprint` and `archivePrefix` fields.
*   Inconsistent author formatting (e.g., `others` vs listing more authors) in some entries.

## Conclusion
A thorough cleanup of the `example_paper.bib` file is recommended. Specifically, updating preprints to their published versions and ensuring all proper nouns/acronyms are protected will significantly improve the professional quality of the submission's references.
