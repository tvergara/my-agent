# Bibliography Audit - Paper 062f9b19

The following citation formatting issues were identified in `icml2026.bib`:

## 1. Missing Capitalization Protection for Acronyms
Several titles contain acronyms that will be rendered in lowercase by most BibTeX styles if not protected with curly braces `{}`:
- `zhou2025reinforcing`: `llms` -> should be `{LLMs}`
- `lee2023rlaif`: `RLAIF`, `RLHF`, `AI` -> should be `{RLAIF}`, `{RLHF}`, `{AI}`
- `bai2022constitutional`: `AI` -> should be `{AI}`
- `tunstall2023zephyr`: `lm` -> should be `{LM}` or `{LM}`
- `zheng2024judging`: `llm`, `mt-bench` -> should be `{LLM}`, `{MT-Bench}`
- `deepseek2025r1`: `LLMs` -> should be `{LLMs}`
- `jiang2025vcrl`: `Vcrl` -> should be `{VCRL}`
- `setlur2025rewarding`: `llm` -> should be `{LLM}`
- `wang2025offline`: `llm` -> should be `{LLM}`
- `greso2025act`: `LLM` -> should be `{LLM}`
- `yao2025optimizing`: `RL` -> should be `{RL}`
- `liao2025rlmr`: `RLMR` -> should be `{RLMR}`
- `liu2025nover`: `NOVER` -> should be `{NOVER}`
- `zhang2025nofree`: `LLM` -> should be `{LLM}`
- `DeepScaleR`: `RL` -> should be `{RL}`
- `empo2025right`: `LLM` -> should be `{LLM}`
- `ttrl2025test`: `TTRL` -> should be `{TTRL}`
- `ren2023selfevaluation`: `NeurIPS` -> should be `{NeurIPS}`
- `sheng2024hybridflow`: `RLHF` -> should be `{RLHF}`

## 2. Inconsistent Title Casing
The bibliography entry titles use a mix of Title Case and sentence case, which leads to inconsistent formatting in the rendered document:
- Title Case: `wang2022self`, `lee2023rlaif`, `bai2022constitutional`, `shao2024deepseekmath`, `yuan2025mars`, `sheng2024hybridflow`.
- Sentence Case: `zhou2025reinforcing`, `rafailov2024direct`, `tunstall2023zephyr`, `zheng2024judging`, `wei2022chain`.

## 3. Proper Noun Protection
Proper nouns in titles like `DeepSeekMath`, `DeepScaleR`, `Zephyr`, and `OlympiadBench` should ideally be protected to ensure they are not lowercased by certain styles.

## 4. Other Issues
- `ttrl2025test`: The journal field is `ArXiv` (Title Case) whereas others use `arXiv preprint arXiv:...` (lower case 'a').
- `ouyang2022training`: The journal `Advances in neural information processing systems` is missing capitalization for `Neural Information Processing Systems`.

These issues can be resolved by consistent use of double curly braces or selective protection of acronyms and proper nouns.
